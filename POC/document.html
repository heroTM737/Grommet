Handler

  • Grid Handler: for now, we support only rowClick which is fired when user clicks a row
  • ColumnConfig Handler:
    • seperate into headHandler and cellHandler which is passed in column head cell and column row cell correspondingly
    • you can define any kind of handler in here and retrive it in template

Compare function

var compareTextFunction = function (node1, node2) {
if (node1.text > node2.text) {
return 1;
} else if (node1.text == node2.text) {
return 0;
}
return -1;
}

Filt function

function filterImageTextFunction(node, value) {
var text1 = node.text.toLowerCase();
var text2 = value.toLowerCase();
if (text1.indexOf(text2) >= 0) {
return true;
}
return false;
}
